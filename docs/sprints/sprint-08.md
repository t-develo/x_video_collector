# Sprint 8: Register Page

## 目的

動画登録ページの実装（URL 入力 → API 呼び出し → 登録結果表示）。

## ブランチ

`feature/sprint8/register-page`

## タスク

### 8.1 登録フォーム

- URL 入力フィールド
- 登録ボタン
- クライアントサイドバリデーション（URL 形式チェック）

### 8.2 登録フロー

1. URL 入力 → バリデーション
2. API POST `/api/videos` 呼び出し
3. ローディング表示
4. 成功 → トースト通知 + 動画一覧へ遷移
5. 失敗 → エラーメッセージ表示

### 8.3 進捗表示

- ダウンロード状態のポーリング（オプション）
- ステータスバッジ（Pending, Downloading, Ready, Failed）

### 8.4 スタイリング

- フォームの CSS
- ボタンの CSS（hover, disabled, loading 状態）
- バリデーションエラー表示

### 8.5 テスト

- フォームレンダリングテスト
- バリデーションテスト
- API 呼び出しテスト（Mock）

## 完了条件

- [ ] 登録フォームが表示される
- [ ] URL バリデーションが動作する
- [ ] API 呼び出しが正しく行われる
- [ ] 成功・エラーのフィードバックが表示される
- [ ] テストがすべてパスする
