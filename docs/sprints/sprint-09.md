# Sprint 9: Video List Page

## 目的

動画一覧ページの実装（カード表示、ページネーション、ソート）。

## ブランチ

`feature/sprint9/video-list-page`

## タスク

### 9.1 動画カードコンポーネント

- サムネイル画像
- タイトル
- ステータスバッジ
- 再生時間
- タグチップ
- 登録日時
- クリックで詳細ページへ遷移

### 9.2 一覧レイアウト

- グリッドレイアウト（CSS Grid）
- レスポンシブカラム数調整

### 9.3 ページネーション

- ページ番号表示
- 前へ / 次へ ボタン
- 1ページあたり件数（20件）

### 9.4 ソート

- 登録日時（新しい順 / 古い順）
- タイトル（昇順 / 降順）
- ソート状態の保持

### 9.5 空状態

- 動画がない場合のメッセージ表示
- 登録ページへの導線

### 9.6 テスト

- カードコンポーネントのレンダリングテスト
- ページネーションのテスト
- API レスポンスのマッピングテスト

## 完了条件

- [ ] 動画一覧がカード形式で表示される
- [ ] ページネーションが動作する
- [ ] ソートが動作する
- [ ] 空状態が適切に表示される
- [ ] テストがすべてパスする
