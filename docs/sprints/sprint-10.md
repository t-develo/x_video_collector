# Sprint 10: Tag & Category UI

## 目的

タグとカテゴリの管理 UI 実装。

## ブランチ

`feature/sprint10/tag-category-ui`

## タスク

### 10.1 タグ管理ページ

- タグ一覧表示
- タグ作成フォーム（名前、色選択）
- タグ編集（インライン編集）
- タグ削除（確認ダイアログ）

### 10.2 カテゴリ管理ページ

- カテゴリ一覧表示
- カテゴリ作成フォーム
- カテゴリ編集
- カテゴリ削除（確認ダイアログ）
- 並び順変更（ドラッグ&ドロップまたは上下ボタン）

### 10.3 動画へのタグ付け

- 動画詳細ページにタグ選択 UI を追加
- タグチップコンポーネント（色付き）
- タグの追加・削除

### 10.4 動画のカテゴリ設定

- 動画詳細ページにカテゴリ選択 UI を追加
- ドロップダウン選択

### 10.5 テスト

- タグ CRUD のテスト
- カテゴリ CRUD のテスト
- タグ付け操作のテスト

## 完了条件

- [ ] タグの CRUD が動作する
- [ ] カテゴリの CRUD が動作する
- [ ] 動画にタグ・カテゴリを設定できる
- [ ] テストがすべてパスする
