# Sprint 11: Search & Filter

## 目的

動画検索とフィルタリング機能の実装。

## ブランチ

`feature/sprint11/search-filter`

## タスク

### 11.1 検索バーコンポーネント

- テキスト入力（タイトル検索）
- デバウンス処理（300ms）
- 検索アイコン + クリアボタン

### 11.2 フィルターパネル

- タグフィルター（複数選択可）
- カテゴリフィルター（単一選択）
- ステータスフィルター（Ready, Pending, Failed）
- フィルターリセットボタン

### 11.3 検索結果表示

- 動画一覧ページと統合
- 検索条件の URL パラメータ反映（ブックマーク可能）
- 結果件数表示
- ハイライト表示（オプション）

### 11.4 サーバーサイド検索

- `SearchVideosUseCase` と API 連携
- 複合条件検索（タイトル + タグ + カテゴリ）

### 11.5 テスト

- 検索バーのデバウンステスト
- フィルター操作のテスト
- API 連携テスト

## 完了条件

- [ ] テキスト検索が動作する
- [ ] タグ・カテゴリ・ステータスでフィルタリングできる
- [ ] 複合条件で検索できる
- [ ] 検索条件が URL に反映される
- [ ] テストがすべてパスする
